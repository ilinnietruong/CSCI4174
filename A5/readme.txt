Exercise 1: Simulation of PlayFair Substation Cipher.
=====================================================================================

Program file: PlayfairSubstitutionCipher.java

*************************************************************************************
Instruction
*************************************************************************************

There are THREE inputs:
1. Enter a string of the secret key.
2. Enter an integer : '1' to encrypt (plaintext to cipher text)
		     '2' to decrypt (cipher text to plaintext).
3. Enter a string of plaintext/cipher text.

Exercise 2 : Simulation of Matrix Transposition Cipher
=====================================================================================

Program file: MatrixTranspositionCipher.java

*************************************************************************************
Instruction
*************************************************************************************

There are FOUR inputs:
1. Enter an integer : '1' to encrypt (plaintext to cipher text)
		     '2' to decrypt (cipher text to plaintext).
2. Enter a string of cipher text/plain text.
3. Enter an integer for the amount of keys.
4. Enter integers of keys to store in the key array.
